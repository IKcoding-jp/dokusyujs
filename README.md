# 独習JavaScript 学習リポジトリ

**独習JavaScript（新版）**（CodeMafia 外村将大 著・翔泳社）を1章から全章こなす学習記録です。
Webフロントエンドエンジニアを目指し、JSの土台を体系的に積み上げることを目的としています。

---

## 学習目的

- Webフロントエンドエンジニアへの転職を目指し、JSの基礎をゼロから積み直す
- 写経・AIに頼るのではなく、**自分で書いて動かす**ことにこだわる
- 学習の継続と積み上げをGitHubで公開・証明する
