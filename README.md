# 独習JavaScript 学習リポジトリ

**独習JavaScript（新版）**（CodeMafia 外村将大 著・翔泳社）を1章から全章こなす学習記録です。
Webフロントエンドエンジニアを目指し、JSの土台を体系的に積み上げることを目的としています。

---

## 学習目的

- Webフロントエンドエンジニアへの転職を目指し、JSの基礎をゼロから積み直す
- 写経・AIに頼るのではなく、**自分で書いて動かす**ことにこだわる
- 学習の継続と積み上げをGitHubで公開・証明する

## 習得キーワード

**ch01:** JavaScriptとは・ブラウザでの動作

**ch02:** `<script>` タグ・外部ファイル読み込み・`console.log` / `window.alert` / `document.write`・大文字小文字の区別・全角半角の区別・セミコロン・コメント・空白・デバッグ基礎

**ch03:** `let` / `const`・キャメルケース・文字列／数値／真偽値／null／undefined・テンプレートリテラル・オブジェクト・ドット記法／ブラケット記法・メソッド・`delete`・型変換・`typeof`

**ch04:** 算術演算子・自己代入演算子・インクリメント／デクリメント（前置・後置）・比較演算子・論理演算子（`&&` / `||` の短絡評価）・falsy値・三項演算子

**ch05:** `if` / `else if` / `else`・`switch`・`while`・`for`・`for...in`・`for...of`・配列ループ・`break`・`continue`・FizzBuzz

**ch06:** 関数宣言・関数式・引数・戻り値・デフォルト引数・ホイスティング・コールバック関数・無名関数・アロー関数

**ch07:** ブロックスコープ・関数スコープ・スクリプトスコープ・グローバルスコープ・モジュールスコープ・レキシカルスコープ・即時関数（IIFE）

**ch08:** `this`・`window` / `globalThis`・メソッド呼び出し時のthis束縛・関数として呼び出した場合のthis・コールバックに渡すとthisが外れる問題（`setTimeout`等）

**ch10:** `Date`オブジェクト（生成・取得・変更）・`confirm`・`setInterval`

**ch11:** 配列メソッド一通り（`forEach` / `map` / `filter` / `reduce` / `find` / `some` / `every` / `flat` / `slice` / `splice` 等）・配列／オブジェクト／関数引数の分割代入・ネストした分割代入・`Map`・`Set`

**ch13:** シングルスレッド・`setTimeout`・イベントループ・タスクキュー（FIFO）・`setTimeout(fn, 0)`でも非同期になる点・コールバック関数・コールバック地獄・`Promise`（`new Promise` / `resolve` / `reject` / `then` / `catch` / `finally`）
